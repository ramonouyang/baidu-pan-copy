📋 如何获取百度网盘Cookie

  1. 打开浏览器，访问 pan.baidu.com 并登录
  2. 按 F12 或 Ctrl+Shift+I (Mac: ⌘+⌥+I) 打开开发者工具
  3. 点击顶部的 Network（网络） 标签
  4. 按 F5 刷新页面
  5. 点击左侧任意请求，在右侧找到 Request Headers
  6. 找到 Cookie: 这一行，复制冒号后面的所有内容
💡 提示：Cookie包含登录凭证（BDUSS等），有效期约30天，请勿分享给他人
